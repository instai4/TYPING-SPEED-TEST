# TYPING_TEST.exe ⌨️

> A sleek, dark NeoBrutalist typing speed test with real-time WPM, accuracy tracking, multiple difficulty modes, and a killer results screen. No fluff — just pure typing practice.

[play it live →](https://instai4.github.io/TYPING-SPEED-TEST/)

---

## 🎯 What Is This?

A fully-featured typing speed test built from scratch. Choose your time limit, start typing, and get detailed stats on your WPM, CPM, accuracy, and mistakes when time runs out. Features a curated set of paragraphs around data science, programming, ML, and tech topics — perfect for a developer audience.

---

## ✨ Features

- ⏱️ **3 time modes** — 30 seconds, 60 seconds, 120 seconds
- 📈 **Real-time WPM & CPM** — updates live as you type
- 🎯 **Live accuracy bar** — color-coded green → yellow → red
- 📊 **Progress bar** — shows how far through the paragraph you are
- 🏁 **Results overlay** — WPM, CPM, accuracy, mistakes, and a personalized tagline
- ⚠️ **Timer warning** — last 10 seconds the timer pulses red
- 🔄 **Auto-scroll** — active character always stays in view
- 💻 **Data science paragraphs** — themed around Python, ML, SQL, GitHub, neural networks
- 🚀 **Entry screen** — animated intro with game rules
- 🖱️ **Custom cursor** — matching the dark NeoBrutalist theme
- 📱 **Responsive** — works on all screen sizes

---

## 🕹️ How to Use

1. Click **"ARE YOU READY?"** on the entry screen
2. Select your time mode: **30s**, **60s**, or **120s**
3. Click the text area or just start typing — the timer begins automatically
4. Type the displayed paragraph as fast and accurately as possible
5. When time runs out, the results screen shows your stats
6. Click **Play Again** or **Try Again** to start a new round

### Color coding while typing

| Color | Meaning |
|-------|---------|
| 🟢 Green | Correct character |
| 🔴 Red | Wrong character |
| 🟡 Yellow (cursor) | Current position |
| ⬜ Gray | Not yet typed |

## 📐 Metrics Explained

| Metric | Formula |
|--------|---------|
| **WPM** | `(chars typed - mistakes) / 5 / elapsed minutes` |
| **CPM** | `chars typed - mistakes` per minute |
| **Accuracy** | `(correct chars / total chars) × 100%` |
| **Mistakes** | Count of incorrectly typed characters |
